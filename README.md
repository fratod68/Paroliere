 # Il Paroliere

  Multilingual vocabulary trainer with SRS (Spaced Repetition System) and AI-powered features.

  ## What is it?

  A vocabulary learning tool designed specifically for language learners who find Duolingo too superficial and Anki too generic.

  ## Features

  - **Structured learning** - Words, Translations, Contexts (not just flashcards)
  - **Bidirectional SRS** - One translation, automatic quizzes in both directions
  - **AI Integration** (Gemini) - Batch translation, spell checking, linguistic analysis
  - **Dashboard** - Track your learning progress
  - **Any language pair** - Learn vocabulary in any language combination
  - **Multi-language UI** - Interface in Italian, German, English
  - **Local data** - SQLite database, no cloud, full control
